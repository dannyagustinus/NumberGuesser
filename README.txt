# Number Guesser
> This project simulates a game that can be played against the computer to see who has the nearest number as the target number

## General Information
- This is a game that makes you input a specific number from 0 - 10 and will compare a random number generated from the computer to a target number that will be randomly generated. Whoever has the nearest number wins and ties can happen when both the user and the computer chooses the same number


## Technologies Used
- Javascript
- HTML
- CSS


## Features
List the ready features here:
- Available to play for multiple rounds
- Step-by-step of how to play shown at the bottom of the website


## Setup
Download all of the necessary documents in this repository and proceed to open the HTML file in your web browser.


## Usage
Click the + or - buttons for choosing a number and then press 'make a guess'. Press the 'next round' button to continue playing
